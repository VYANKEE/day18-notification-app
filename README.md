###Day 18 of my 45 Days Coding Challenge


###Project Name: The Vintage Ledger


###Description
The Vintage Ledger is a unique web application designed to simulate a classic, old-school digital archive. What sets this project apart is its primary focus on auditory interaction. The application is built around an immersive Voice Notification feature that acts as the main feedback mechanism for the user. Every significant action within the terminal is accompanied by a specific robotic voice confirmation, creating a vintage mainframe experience. This combines with a retro visual aesthetic to create a completely immersive environment.

Key Features
Voice-Driven Interface: The core feature of the application. A built-in robotic voice assistant verbally confirms every protocol, such as logging in, applying for jobs, or running security checks, making the experience feel alive and interactive.

Immersive Vintage UI: Designed with aged paper textures, classic typography (Playfair Display and Courier Prime), and retro styling.

Mechanical Sound Effects: Complements the voice system with realistic typewriter mechanical sounds for every interaction.

Secure Authentication: Fully functional Login and Registration system to secure access to the terminal.

Interactive Control Panel: Users can trigger simulated tasks like Job Applications and Inventory Checks, which instantly trigger the voice feedback system.

Real-time Correspondence: Actions are instantly logged into a digital notebook with timestamped entries.

Technology Stack
Frontend: React.js (Vite)

Styling: CSS3 (Custom Vintage Theme)

Animations: Framer Motion

Icons: Lucide React

Deployment: Firebase Hosting

How to Run Locally
Clone this repository to your local machine.

Open the terminal in the project folder.

Run the command: npm install

Start the development server using: npm run dev

Open the local link provided in the terminal to view the app.
